2018-05-01 | 技术
|

前言

在这些年的开发过程中，基本上都是直接使用ssh或者ssm框架，直接调用去实现业务功能。
最近在开源中国上看到一遍文章初学 Java Web 开发，请远离各种框架，从 Servlet 开发， 然后我开始反思， 好像离开了这些框架，忘了怎么去搭建一个基本后端链接，真的原生的web的知识了解的不是很深，从现在开始着手,说干就干
